# 일정 관리 앱 API 서버

## 프로젝트 소개

해당 프로젝트는 일정을 관리하는 앱의 API 서버를 프로그램을 제작하는 프로젝트이다.

## 프로젝트 목적

- API 명세서 작성 학습
- Spring MVC 학습
- Spring JDBC Template 학습
